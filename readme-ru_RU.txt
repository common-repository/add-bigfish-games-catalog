== Installation ==

1. Создать папку 'images' в корневой директории вашего блога и установите права на запись (chmod 777) на папку 'images'.
2. Залейте папку со скриптом в папку wp-content/plugins.
3. Активируйте плагин в административной панели.
4. Откройте Настройка -> Add BigFish Games Catalog и начинайте добавление игр.
