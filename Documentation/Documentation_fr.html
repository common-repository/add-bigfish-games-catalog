Installation

  1. Créez une chemise 'les images' dans le blog d'annuaire de racine et mettez écrivent des permissions (chmod 777) 'aux images'.
  2. Téléversez le plein annuaire dans votre annuaire wp-content/plugins.
  3. Activez l'enfichable à la page d'administration enfichable.
  4. Ouvrez la page de configuration enfichable, qui est localisée sous les Options-> Ajoutent le Catalogue de Jeux BigFish et construisent le catalogue.
